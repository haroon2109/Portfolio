# Mohamed Haroon | Neo Brutalist Portfolio 🚀

A high-impact, interactive personal portfolio built with a **Neo Brutalist** design philosophy. Designed for visual excellence, this portfolio showcases my expertise in **AI & Data Science** and **Full-Stack Development**.

## ✨ key features

### 🎨 neo brutalist design
- **Vibrant Palette**: Bold Yellow, Pink, and Cyan high-contrast theme.
- **Tough Aesthetics**: Consistent 4px black borders and 8px hard offset shadows.
- **Aggressive Typography**: Heavy-weight "Inter" headers with outlined accent text.

### 🖱️ advanced interactivity
- **Custom Inverting Cursor**: A smooth "Lerp" cursor with `mix-blend-mode: difference` for real-time color inversion.
- **Magnetic Buttons**: Elements that pull toward your mouse for a tactile experience.
- **Audio Feedback**: Built-in "Pop" sound effects using the **Web Audio API** on interactions.
- **Dynamic Project Filtering**: Instant categorization (AI/ML, Web, Tools) with smooth transitions.

### 📊 live data integration
- **GitHub Stats API**: Real-time stats display for stars, commits, and activity tracking.
- **Interactive Timeline**: A bold, linear journey of my education and internships.

## 🛠️ tech stack

- **Frontend**: Vanilla HTML5, CSS3 (Modular Design System), JavaScript (ES6+).
- **Icons**: Font Awesome 6.4.
- **Typography**: Google Fonts (Inter, Dancing Script).
- **Interactions**: Intersection Observer API, Web Audio API, Mouse Event Lerping.

## 📁 project structure

- `index.html`: Core structure and semantic layout.
- `style.css`: Neo Brutalist design tokens and animation logic.
- `script.js`: Interactive features, theme handling, and filtering logic.
- `assets/`: Optimized project imagery and branding assets.
